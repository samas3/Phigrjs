## 使用方法
1. 使用你熟悉的HTTP服务器来提供文件服务，例如：
   - Python的http.server模块：
     ```bash
     python -m http.server 8000
     ```
   - Node.js的http-server：
     ```bash
     npx http-server
     ```
   - 或其他任何HTTP服务器
2. 在浏览器中访问服务器提供的地址即可开始游戏
